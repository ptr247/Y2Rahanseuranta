# Esittely
Kulutuskäyttäytymisen visualisointiin tarkoitettu ohjelma.
Käyttäjä voi ladata ohjelmaan tiedoston omista tilitapahtumistaan csv-muodossa 
ja luokitella tapahtumia suurempiin kokonaisuuksiin.

## Tiedosto- ja kansiorakenne
Kansiossa src sijaitsee projektin koodin sisältävät .py tiedostot.

## Asennusohje
Ohjelma tarvitsee kirjastoja PyQt5 ja PyQtchart.
Asennetaan "pip install PyQt5" ja "pip install PyQtChart"

## Käyttöohje
Ohjelma ajetaan main.py:n avulla.
Main.py:n voi suorittaa esimerkiksi Windowsin komentokehotteessa navigoimalla cd:n avulla oikeaan tiedostosijaintiin
ja sen jälkeen kirjoittamalla komentokehotteeseen "python main.py".
